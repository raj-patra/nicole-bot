Nicole - A Telegram Bot¶

INTRODUCTION¶

Nicole, is a conversational chat-bot made for Telegram using Python and AIML. She has many interactive menus which are linked to various free APIs for recreational purposes. She also has a brain made out of AIML categories which provides it with its chatterbot abilities.

Nicole can also be added to your private telegram groups, but she will have no access to any of your messages unless mentioned or replied to by any of the group members.

Interaction menus¶

Interaction menus are recreational options from Nicole which lets her users to connect to various web services, and extract fun content right from the web.

Main menu¶

  • Image APIs - Contains a list of APIs which mainly deals with images.

  • Text APIs - Contains a list of APIs which mainly deals with texts.

  • Services & Utilities - Contains a list of tools that an user can take help of.

Image menu¶

The image menu contains a collection of free services/APIs that mainly deal with images. It can be as simple as extracting a random meme from reddit or as complex as generating a artificial face from an AI model.

COMMANDS¶

The Nicole bot has some handy commands which can be best utilised in a private group.

  • /start - Initiates the conversation with the bot.

  • /menu - Sends out the interaction menu

  • /slap - The infamous discord slap from Dank Memer. Mentioning group members with this will slap them for you.

  • /roast - Yep, you’ve guessed it. It roasts your group members from your behalf, you just need to mention them.

  • /dev - Last but not least, the developer’s contact. For any queries or suggestions or feedback.

NOTE - The said bot doesn’t record or store any conversations, nor does it use the messages to train it’s brain. The brain file is static and trained beforehand.
